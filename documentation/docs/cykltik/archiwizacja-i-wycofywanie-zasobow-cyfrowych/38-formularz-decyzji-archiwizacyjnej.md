# Formularz decyzji archiwizacyjnej

## Cel

Celem formularza jest zapewnienie jednolitego zapisu decyzji archiwizacyjnej. Formularz ma umożliwiać odtworzenie, kto podjął decyzję, czego dotyczyła, na jakiej podstawie została podjęta, jaki status otrzymał zasób i jak będzie dostępny.

## Zakres

Formularz stosuje się do dokumentów elektronicznych, treści WWW i BIP, załączników, multimediów, danych z systemów, zasobów projektowych, paczek archiwalnych i zasobów historycznych.

## Zasady

Formularz należy wypełnić przed nadaniem statusu archiwalnego, przekazaniem zasobu, wycofaniem z publikacji z zachowaniem zasobu albo podjęciem decyzji o dalszym przechowywaniu. Formularz nie zastępuje procedur archiwum zakładowego, ale powinien być z nimi zgodny.

### Opis pól

| Pole | Sposób wypełnienia |
| --- | --- |
| ID | Unikalny identyfikator zasobu, wpisu, dokumentu, paczki albo decyzji. |
| Nazwa | Tytuł zasobu albo krótki opis pozwalający go rozpoznać. |
| URL | Adres publiczny albo wewnętrzna lokalizacja, jeżeli zasób nie był publiczny. |
| Właściciel | Komórka albo osoba odpowiedzialna za treść, dane albo dokument. |
| Kategoria | Kategoria archiwalna, klasa JRWA albo informacja, że kwalifikacja nie dotyczy. |
| Decyzja | Archiwizować publicznie, archiwizować z dostępem na wniosek, przekazać, poprawić przed archiwizacją, przeznaczyć do dalszej kwalifikacji albo inna decyzja. |
| Dostęp | Publiczny, archiwalny publiczny, na wniosek, ograniczony, wewnętrzny albo brak publicznego dostępu z podaniem podstawy. |
| Data | Data podjęcia decyzji. |
| Odpowiedzialny | Osoba odpowiedzialna za wykonanie decyzji. |
| Uwagi | Ryzyka, warunki, opinie IOD, archiwum zakładowego, IT, koordynatora dostępności albo obsługi prawnej. |

## Ryzyka

Niewypełnienie formularza powoduje brak śladu decyzyjnego. Największe ryzyka to nieustalenie właściciela, kategorii, sposobu dostępu, statusu po decyzji i osoby odpowiedzialnej za wykonanie.
