# Lista kontrolna wycofania treści

## Cel

Celem listy jest potwierdzenie, że treść może zostać wycofana z publikacji bez naruszenia obowiązków informacyjnych, archiwalnych, ochrony danych osobowych i dostępności cyfrowej.

## Zakres

Lista dotyczy stron WWW, wpisów BIP, dokumentów PDF, załączników, komunikatów, formularzy, multimediów, treści projektowych i treści otrzymanych od innych podmiotów.

## Zasady

Odpowiedzi należy oznaczać jako: TAK, NIE albo NIE DOTYCZY. Odpowiedź NIE przy punkcie blokującym oznacza, że treść nie może zostać wycofana do czasu zakończenia analizy albo podjęcia udokumentowanej decyzji.

## Szczegóły

| Obszar | Pytanie | TAK | NIE | NIE DOTYCZY | Punkt blokujący |
| --- | --- | --- | --- | --- | --- |
| Identyfikacja | Czy ustalono tytuł, URL, typ treści, datę publikacji i właściciela? |  |  |  | Tak |
| Informacja publiczna | Czy oceniono, czy treść jest informacją publiczną? |  |  |  | Tak |
| Informacja publiczna | Czy określono, czy informacja będzie dostępna publicznie, na wniosek albo w aktach sprawy? |  |  |  | Tak |
| BIP | Czy ustalono, czy treść była opublikowana w BIP? |  |  |  | Tak |
| BIP | Czy administrator BIP uczestniczył w decyzji albo wykonaniu czynności? |  |  |  | Tak, jeżeli dotyczy |
| Dane osobowe | Czy sprawdzono, czy treść zawiera dane osobowe? |  |  |  | Tak |
| Dane osobowe | Czy IOD ocenił ryzyko albo podstawę ograniczenia publikacji, jeżeli dane są istotne? |  |  |  | Tak, jeżeli dotyczy |
| Archiwizacja | Czy zabezpieczono kopię źródłową i metadane? |  |  |  | Tak |
| Archiwizacja | Czy ustalono kategorię archiwalną albo powiązanie ze sprawą? |  |  |  | Tak, jeżeli dotyczy dokumentacji |
| Dostęp alternatywny | Czy wskazano sposób uzyskania informacji po wycofaniu? |  |  |  | Tak |
| Dostęp alternatywny | Czy przygotowano komunikat, przekierowanie albo kontakt dla użytkownika? |  |  |  | Tak, jeżeli użytkownik może szukać treści |
| Dostępność | Czy oceniono, czy wycofanie nie służy obejściu obowiązków dostępności? |  |  |  | Tak |
| Decyzja | Czy decyzję podjął właściciel zasobu albo właściwe kierownictwo? |  |  |  | Tak |
| Dokumentowanie | Czy zapisano powód, datę, status po decyzji i osobę odpowiedzialną? |  |  |  | Tak |

Punkty blokujące obejmują brak właściciela, brak oceny informacji publicznej, brak oceny BIP, brak oceny danych osobowych, brak zabezpieczenia kopii źródłowej, brak decyzji i brak określenia dalszego dostępu.

## Ryzyka

Największe ryzyka to usunięcie informacji publicznej, ukrycie treści BIP bez podstawy, utrata dokumentu sprawy, naruszenie RODO, brak dostępu alternatywnego i brak możliwości wykazania, dlaczego treść została wycofana.
