# Zarządzanie archiwizacją i wycofywaniem zasobów cyfrowych

Pakiet opisuje zarządzanie końcem cyklu życia zasobów cyfrowych z perspektywy dostępności cyfrowej, trwałości informacji publicznej, archiwizacji dokumentacji i bezpieczeństwa danych. Nie jest pełnym systemem archiwizacji cyfrowej. Porządkuje decyzje organizacyjne, które trzeba podjąć przed archiwizacją, migracją, wycofaniem treści albo zamknięciem systemu.
