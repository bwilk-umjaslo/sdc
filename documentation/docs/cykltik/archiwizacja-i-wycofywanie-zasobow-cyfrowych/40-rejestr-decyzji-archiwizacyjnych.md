# Rejestr decyzji archiwizacyjnych

## Cel rejestru

Rejestr decyzji archiwizacyjnych służy do ewidencjonowania decyzji dotyczących zachowania, oznaczenia, przekazania, udostępniania albo dalszej kwalifikacji zasobów cyfrowych. Rejestr powinien umożliwiać odtworzenie, kto podjął decyzję, jakiego zasobu dotyczyła, jaki był status przed decyzją, jaki jest sposób dalszego dostępu i gdzie znajduje się dokument decyzji.

Rejestr uzupełnia [Formularz decyzji archiwizacyjnej](./38-formularz-decyzji-archiwizacyjnej.md) i powinien być stosowany razem z [Procedurą archiwizacji zasobów](./10-procedura-archiwizacji-zasobow.md).

## Zasady prowadzenia

Rejestr należy prowadzić w sposób zapewniający trwałość wpisów, kontrolę zmian, możliwość wyszukiwania i powiązanie z dokumentacją sprawy, jeżeli zasób jest dokumentem albo elementem akt sprawy.

Wpis w rejestrze powinien powstać przed wykonaniem decyzji albo najpóźniej bezpośrednio po jej wykonaniu. Nie należy traktować wpisu technicznego w CMS, BIP albo repozytorium plików jako pełnego rejestru decyzji, jeżeli nie zawiera wymaganych danych.

Za prowadzenie rejestru odpowiada właściciel procesu archiwizacji wskazany przez podmiot. W zależności od organizacji może to być archiwum zakładowe, komórka koordynująca zarządzanie zasobami cyfrowymi albo właściciel systemu rejestrowego. Właściciel zasobu odpowiada za prawidłowość informacji merytorycznych, a archiwum zakładowe za zgodność z kwalifikacją archiwalną.

## Opis kolumn

| Kolumna | Sposób wypełnienia |
| --- | --- |
| ID decyzji | Unikalny identyfikator decyzji w rejestrze. |
| Data | Data podjęcia decyzji albo zatwierdzenia decyzji. |
| Nazwa zasobu | Nazwa dokumentu, strony, pliku, zbioru danych, paczki albo innego zasobu. |
| Typ zasobu | Dokument, treść WWW, treść BIP, multimedia, dane, systemowy eksport, zasób projektowy albo inny typ. |
| Status przed | Status zgodny z [Modelem statusów zasobów](./07-model-statusow-zasobow.md). |
| Decyzja | Archiwizować, oznaczyć jako archiwalny, przekazać, udostępniać na wniosek, poprawić, migrować, przeznaczyć do kwalifikacji albo inna decyzja. |
| Kategoria archiwalna | Kategoria A, kategoria B, klasa JRWA albo informacja, że nie dotyczy. |
| Sposób dostępu | Publiczny, archiwalny publiczny, na wniosek, wewnętrzny, ograniczony albo brak publicznego dostępu z podstawą. |
| Osoba odpowiedzialna | Osoba albo rola odpowiedzialna za wykonanie decyzji. |
| Link do dokumentu decyzji | Odnośnik do formularza, sprawy, protokołu, wpisu w EZD albo innego dokumentu decyzji. |
| Uwagi | Ryzyka, warunki, opinie, odstępstwa, termin przeglądu albo informacje o dostępie alternatywnym. |

## Powiązania

Rejestr powinien być stosowany z [Modelem decyzyjnym](./08-model-decyzyjny.md), [Modelem ryzyk](./09-model-ryzyk.md), [Mapą odpowiedzialności](./30-mapa-odpowiedzialnosci.md), [Listą kontrolną archiwizacji](./35-lista-kontrolna-archiwizacji.md), [Standardem metadanych](./16-standard-metadanych.md) i [Dokumentowaniem procesu wycofania](./26-dokumentowanie-procesu-wycofania.md).
