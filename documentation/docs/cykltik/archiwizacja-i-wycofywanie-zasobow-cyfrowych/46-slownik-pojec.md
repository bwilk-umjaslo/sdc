# Słownik pojęć

## Cel

Celem słownika jest ujednolicenie pojęć stosowanych w zaleceniu. Definicje mają charakter operacyjny i służą stosowaniu procedury w podmiocie publicznym.

## Pojęcia

| Pojęcie | Definicja |
| --- | --- |
| Archiwizacja cyfrowa | Zespół czynności służących zachowaniu zasobu cyfrowego z treścią, metadanymi, kontekstem, integralnością i możliwością odczytu w czasie. |
| Zasób cyfrowy | Dokument, treść, dane, plik, multimedia, rekord, formularz, wpis, eksport albo inny obiekt przechowywany lub udostępniany w postaci cyfrowej. |
| Treść archiwalna | Treść, która nie pełni już funkcji bieżącej, ale została zachowana i może być dostępna publicznie albo na wniosek ze względu na wartość informacyjną, dokumentacyjną albo historyczną. |
| Wycofanie | Kontrolowane zakończenie publicznej dostępności treści albo eksploatacji systemu po ocenie danych, dokumentacji, dostępności, retencji i sposobu dalszego dostępu. |
| Migracja danych | Przeniesienie danych, dokumentów, metadanych i relacji do innego systemu, formatu albo repozytorium z potwierdzeniem możliwości odtworzenia. |
| Retencja | Okres i zasady przechowywania dokumentacji, danych albo zasobów wynikające z przepisów, JRWA, potrzeb dowodowych albo obowiązków organizacyjnych. |
| Brakowanie | Formalne niszczenie albo usuwanie dokumentacji kategorii B po upływie okresu przechowywania i spełnieniu wymaganych warunków. |
| JRWA | Jednolity rzeczowy wykaz akt, służący klasyfikacji dokumentacji i ustalaniu kategorii archiwalnej oraz okresu przechowywania. |
| Kategoria A | Kategoria materiałów archiwalnych przeznaczonych do trwałego przechowywania. |
| Kategoria B | Kategoria dokumentacji niearchiwalnej przechowywanej przez określony czas, po którym może podlegać brakowaniu zgodnie z procedurą. |
| Archiwum zakładowe | Komórka albo funkcja odpowiedzialna za przejmowanie, przechowywanie, ewidencjonowanie i udostępnianie dokumentacji w podmiocie. |
| Archiwum państwowe | Właściwa jednostka państwowej sieci archiwalnej nadzorująca postępowanie z materiałami archiwalnymi w zakresie wynikającym z przepisów. |
| Metadane | Dane opisujące zasób, w szczególności identyfikator, nazwę, właściciela, daty, status, format, źródło, kategorię, lokalizację i powiązanie ze sprawą. |
| Integralność | Właściwość zasobu potwierdzająca, że jego treść, dane, metadane albo relacje nie zostały nieuprawnienie zmienione. |
| Autentyczność | Możliwość wykazania pochodzenia, autorstwa, kontekstu i wiarygodności zasobu. |
| Dostęp alternatywny | Sposób przekazania informacji użytkownikowi w innej formie, gdy zasób nie jest w pełni dostępny cyfrowo albo nie może zostać niezwłocznie dostosowany. |
| Zasób historyczny | Zasób zachowany ze względu na znaczenie dla historii sprawy, projektu, działania podmiotu albo informacji publicznej. |
| Właściciel zasobu | Osoba albo komórka odpowiedzialna za znaczenie merytoryczne zasobu, aktualność, decyzję o statusie i sposób dalszego dostępu. |
| Właściciel systemu | Osoba albo komórka odpowiedzialna za system, w którym zasób jest przechowywany, przetwarzany, publikowany, migrowany albo wycofywany. |

## Zasady stosowania

Jeżeli pojęcie ma znaczenie prawne w przepisach albo dokumentacji wewnętrznej podmiotu, należy stosować definicję prawną albo organizacyjną. Definicje słownika nie zastępują JRWA, instrukcji kancelaryjnej, instrukcji archiwalnej ani przepisów szczególnych.
